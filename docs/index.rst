Python Pipeline 
===============

Pipeline is a Python library to help write clean code utilizing a reusable
pipeline pattern.

It encourages readability, testability, cohesion, concurrency, and strict
limits to communication. And we provide tools for developers building
applications with Pipelines.

Continue reading to learn more and get involved on `Github
<http://github.com/alexlemann/pipeline/>`_
